This page provides an overview of how the content on this site is generated and maintained, specifically with regard to large language models (LLMs, often referred to as "AI").

Since its inception, this site is and has been completely human-generated and human-maintained:

* LLMs are not used to generate any of the text on this site. All text content is written by a human without the use of an LLM.
* Text content is produced in [Markdown][1] using a text editor. No LLM integration of any kind is used in the text editor.
* Markdown syntax and formatting are verified using [Markdownlint][2], which runs locally on my system(s). No LLM is involved.
* Writing style is checked using [Vale][3], which runs locally on my system(s). No LLM is involved.
* Link checking/verification is performed locally on my system(s). No LLM is involved.
* LLMs are not used to generate any graphics or diagrams on this site. All graphic content is created by a human without the use of an LLM.
* LLMs are not used to generate any of the example code on this site. All code is written by a human without the use of an LLM.

There is no plan to start using LLMs for content generation of any kind.

[3]: https://vale.sh/
